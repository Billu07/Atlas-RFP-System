Configuration Guide

Quick Start Guide

  1. Set up your Airtable base using the schema in AIRTABLE_SETUP.md
  2. Copy config.example.js to config.js
  3. Add your Airtable Base ID and Personal Access Token
  4. Change the admin password in config.js
  5. Deploy to Vercel or your hosting platform

Creating a New RFP

  1. Click "Create New RFP" in the RFPs section
  2. Fill in all required fields
  3. Set status to "Active" when ready to publish
  4. Share the public link with invited vendors

Reviewing Submissions

  1. Go to Submissions section
  2. Click on any submission to view details
  3. Update review status and add internal notes
  4. Use star ratings to compare vendors
